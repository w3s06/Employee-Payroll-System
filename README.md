# Employee Payroll System (Arduino / C++)

A microcontroller-based Employee Payroll System developed using Arduino C++.  
The program reads predefined employee information (name, ID, job title, grade, salary, pension) and displays it on a connected hardware output. This project demonstrates embedded programming, data structuring, and real-time information display using the Arduino environment.

---

## ✨ Features
- Displays employee details including:
  - Employee ID  
  - Job title and grade  
  - Salary  
  - Pension information
- Uses Arduino-friendly C++ code and serial output or LCD (depending on configuration).
- Simple, lightweight implementation suitable for embedded systems coursework.

---

## 🧰 Tech Stack
- **Language:** Arduino C++  
- **Tools:** Arduino IDE  
- **Hardware:** Arduino board (e.g., Uno/Nano)  
- **Concepts:** Embedded systems, microcontroller programming, serial display

## 🚀 How to Run
1. Open the `.ino` file in **Arduino IDE**.  
2. Connect your Arduino board.  
3. Select the correct board and port:  
   *Tools → Board → Arduino Uno (or your specific model)*  
4. Upload the program.  
5. Open the Serial Monitor (or LCD display if used) to view employee details.
